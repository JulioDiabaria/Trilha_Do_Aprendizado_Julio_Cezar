Hiperlinks

Hiperlink ou hiperligação é qualquer elemento de um hiper texto (páginas web) que façam referência a outro texto ou a outra parte deste texto. Ou seja, qualquer área clicável de uma página web é um hiperlink. Um hiperlink pode também ser referido como um linkl ou hipertexto.

Para criar um hiperlink deve ser usado o seguinte código: < href="local de referência" > < target="_blank">

O texto que conterá o hiperlink para o redirecionamento não precisa necessáriamente ser um texto, pode ser qualquer outro elemento do documento HTML, como por exemplo uma imagem.

Atributos

href
É o atributo onde fica o link para o qual o usuário será redirecionado ao clicar.
lugar onde estará o link
É o que fica visível ao usuário e, ao passar o cursor por cima, da a opção de clicar no link.
target
É o atributo que especifica onde o link sera aberto, possuindo alguns valores, como:
_self
Modo padrão, mesmo que o atributo target não seja especificado, será esse o valor utilizado. Abrirá o ducumento na mesma janela em que foi clicado.
_blank
Abre o link em uma nova guia.
